# 混沌系统敏感文本信息加密算法设计 python1032

## 项目概述

本项目是基于Python和Django框架的Web应用，旨在通过混沌系统实现敏感文本信息的加密。系统采用B/S架构，以MVC模式进行设计，后端数据库使用MySQL。

## 技术栈

- **开发语言**：Python
- **框架**：Django
- **数据库**：MySQL
- **架构**：B/S，MVC
- **系统环境**：Windows
- **开发环境**：PyCharm

## 功能模块

### 系统角色

系统设计包括两个角色：用户和管理员。

#### 前台系统

- **注册与登录**：用户可以创建账户，登录后使用系统全部功能。
  - 用户名、密码输入
  - 账户创建
- **文本加密**：系统利用混沌技术对敏感信息进行加密。
  - 明文转换为密文
- **加密文本解密**：用户可通过秘钥将加密后的文本解密为原始明文。
  - 输入正确秘钥
- **加密管理**：
  - 查看加密文本记录
  - 下载加密文本
  - 删除加密文本

#### 后台管理系统

- **用户管理**：
  - 查看用户信息
  - 添加用户
  - 删除用户
  - 修改用户信息
- **加密信息管理**：
  - 查看用户加密信息
  - 搜索加密信息
  - 修改加密信息
  - 删除加密信息
- **系统信息展示**：
  - 加密数量统计
  - 用户数量统计
  - 系统运行状态展示

## 运行环境

- 操作系统：Windows
- Web服务器：建议使用与Django兼容的服务器，如Gunicorn或uWSGI
- 数据库：MySQL
